# Favicon Generator Instructions

Since we're using the logo.png as our favicon, you may want to create optimized versions:

## Recommended Favicon Sizes:
- favicon.ico (16x16, 32x32, 48x48 pixels combined)
- favicon-16x16.png
- favicon-32x32.png
- apple-touch-icon.png (180x180)
- android-chrome-192x192.png
- android-chrome-512x512.png

## Tools to Convert:
1. Use online favicon generators like favicon.io
2. Use image editing software to resize logo.png
3. Upload logo.png to https://realfavicongenerator.net/

## Current Setup:
We're using logo.png for all favicon purposes. Modern browsers will automatically resize as needed.
